Design Summary

Features and Functions

This website is an e-commerce solution for a made up restaurant(WaRu restaurant) near USIU-Africa.It features a home page , Menu page, Reservation page, About page and Contact Us page

Potential customers can navigate to the site and immediately reserve a table and time using the Reservation page. The Menu page displays the restaurant's offerings and prices to make it easier for the customer to budget.

For any information/feedback the customer can reach us via the Contact Us page and can read information about the founding of the restaurant and opening/closing hours and location of the restaurant on the About Page

Design Choices

I employed use of HTML/CSS/Javascript for the frontend part of the website. For additional styling and responsiveness I employed use of Bootstrap Framework

The responsiveness.css stylesheet was used for responsiveness of the site and third party(classic.css/classic.date.css/classic.time.css) was used for enabling the customer to select a date and time during his/her reservation

The font-awesome.css stylesheet was used for various custom fonts on the website

Similarly Javascript was used to validate forms sent, in loading images , and custom javascript used by Bootstrap Framework for purposes of button changing , the carousel, transitions, animations etc

Lastly, I employed use of a bit of php for simple validation of the forms. However, I did not implement server side architecture for this website so the forms aren't actually submitted to a valid email
